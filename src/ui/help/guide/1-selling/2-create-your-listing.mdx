## Create your listing

Creating your listing just takes a few simple steps. Get started <AnchorButton target="_blank" href="/list" className="inline-flex" kind="tertiary">here</AnchorButton>.

### <a id="region" href="#region" className="button">Region</a>

A game will have a region code depending on where in the world it was released. Some games were only released in one country, but most will have been released worldwide.

Most retro consoles are region locked, meaning a console from one region can only play games from the same region.

Telling us your region code helps to ensure we show your listing to the right buyers. Some people also like to collect all regions of a game, or all games from a specific region.

If you're not sure what region your game is, you can skip this step.

### <a id="condition" href="#condition" className="button">Condition</a>

Condition is a highly subjective property so hopefully these descriptions can help you decide what to pick

#### Mint

<img src={mint} alt="mint condition" />

A mint condition game is sealed in the box, never been used. The holy grail of game conditions üèÜ

#### Like new

<img src={likeNew} alt="like new condition" />

The game is boxed with instructions, there is no visible wear on the game or box. This is often referred to as CIB or "Complete In Box"

#### Used

<img src={used} alt="used condition" />

Good condition. The game may have a few marks, the instructions are looking a bit dog-eared, your mum threw the box away 10 years ago.

This is the most common state for older games.

#### Poor

<img src={poor} alt="poor condition" />

A poor condition game has suffered some severe wear. The game itself should still work, however.

Keep in mind that there might be some cross over. A "used" game might be unboxed without instructions, but a "poor" game might have both but in poor condition. Keep this in mind when deciding what condition to pick.

We will ask about whether you have the instructions/box in the next step.
